# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

A Rust codebase for studying functional programming concepts (Rustを用いた関数型プログラミングの勉強).

## Common Commands

```bash
cargo build          # Build
cargo test           # Run all tests
cargo test <name>    # Run a single test by name
cargo run            # Run the binary
cargo clippy         # Lint
cargo fmt            # Format code
```


## 学習コンテンツの記載の方針

- サンプルプログラムを載せることでよりわかりやすくなるようならプログラムを載せてください。
- サンプルプログラムはmdファイルに載せるとともにcrate/フォルダに実行可能なプログラムも作るようにして下さい。
