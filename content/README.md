# Rustで学ぶ関数型プログラミング

Rust を使って関数型プログラミングの概念を学ぶ学習リポジトリです。

---

## 関数型プログラミングを知らない方へ

「関数型プログラミング（FP）」という言葉を初めて聞いた方でも読み進められるように書かれています。

**[第0章](00_intro.md)** から読み始めることをおすすめします。日常的なアナロジーを通じて、FPの考え方を直感的に掴めます。

### この資料で学べること

- 純粋関数・不変性・副作用とは何かを理解する
- Rust の `iter()` や `match` を関数型スタイルで使いこなす
- 小さな関数を組み合わせてバグの少ないコードを書く力

### 前提知識

- Rust の基本的な文法（変数宣言・関数・`if`/`for`）が読める程度で十分です
- 関数型プログラミングの知識は不要です

## サンプルコード

各章のサンプルコードは [`crate/`](https://github.com/Moto1829/study-of-functional-programming/tree/main/crate) ディレクトリで実行できます。

```bash
cd crate
cargo test          # 全テスト実行
cargo test -p closures  # 特定章のテスト実行
```
